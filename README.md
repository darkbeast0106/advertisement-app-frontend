# Apróhirdetés alkalmazás frontend

Backend: [https://github.com/darkbeast0106/advertisement-app-backend](https://github.com/darkbeast0106/advertisement-app-backend)

## Követelmények

- NodeJS 18

## Telepítés lépései

- Csomagok telepítése
  
  ```sh
  npm install
  ```

- .env állomány létrehozása
  
  ```sh
  cp .env.example .env
  ```

- Futtatás (local)
  
  ```sh
  npm run dev
  ```
